# pySide6-url-request

![](https://i.imgur.com/l9XIKlq.png)

Extraer respuestas de urls dentro de una interfaz gráfica con pySide6

## Librerías necesarias

- PySide6
- requests
- bs4

```bash
pip install -r requirements.txt
```

## Caracteristicas
- Extraer el header y contenido de paginas web
